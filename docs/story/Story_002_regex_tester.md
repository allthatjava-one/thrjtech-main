[ ] I want this app to have a feature that test Regular Expressions (RegEx). The feature should allow users to input a RegEx pattern and a test string, and then display the matches found in the test string based on the RegEx pattern. The UI should be simple and user-friendly, with clear input fields for the RegEx pattern and test string, and a section to display the results. ## UI Change
- Add a new section for the RegEx tester with two input fields: one for the RegEx pattern and one for the test string.
- Include a "Test" button that users can click to execute the RegEx test.
- Display the results in a clear and organized manner, showing the matches found in the test string.
- Provide a list of common RegEx patterns as clickable examples that populate the RegEx input field when clicked.
- Look and feel should be similiar to the existing tools in the app(Json-Formatter), with a clean and modern design that is consistent with the overall UI.

## Functionality
- When the user inputs a RegEx pattern and a test string and clicks the "Test" button, the app should process the RegEx pattern against the test string and display the matches.
- The app should handle invalid RegEx patterns gracefully, providing an error message to the user if the pattern is not valid.
- Clicking on a common RegEx pattern example should populate the RegEx input field with that pattern
- The results section should update dynamically based on the input and show all matches found in the test string.

## User Experience
- The input fields for the RegEx pattern and test string should be clearly labeled and easy to use.
- The "Test" button should be prominently displayed and easy to click.
- The results should be displayed in a way that is easy to read and understand, with matches highlighted or listed clearly.
- The common RegEx pattern examples should be easily accessible and clearly indicate that they are examples that can be clicked to populate the input field.

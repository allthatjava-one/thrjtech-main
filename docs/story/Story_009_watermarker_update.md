Enhancing Watermarker.

In this story, we will enhance the watermarker by adding a new feature that allows users to customize the watermark text and its position on the image. This will provide users with more flexibility and control over how their watermarks are displayed.

# New Feature: Customizable Watermark Position
- We will add an option for users to choose the position of the watermark on the image. The available positions will include:
  - Top Left
  - Top Right
  - Bottom Left
  - Bottom Right
  - Center
  - Default (Center and diagonal)
- Users will be able to select their desired position from a dropdown menu in the watermarker settings.
- The watermark will be dynamically positioned based on the user's selection, ensuring that it is displayed correctly on the image regardless of its size or orientation.
- Repeated Option can be only selected if the position is set to Default, allowing users to have multiple watermarks displayed in a diagonal pattern across the image. If other option is selected, the repeated option will be disabled to prevent confusion.

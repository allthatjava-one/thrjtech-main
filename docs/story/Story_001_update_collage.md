This is a story about updating a collage feature.

## UI Change
- Have a file dropzone for users to upload images.
- Display the uploaded images in a grid layout. Also Upload grid layout adjust to column and row count.
- Allow users to rearrange the images by dragging and dropping them within the grid.
- Provide options for users to delete images from the collage.
- Include a preview button to see the collage before saving.
- Have a color choice for the background of the collage. Once the user selects a color, the background of the collage will change accordingly.

## Functionality
- When a user uploads an image, it should be added to the collage grid.
- The user should be able to rearrange the images by dragging and dropping them within the grid
- When a user deletes an image, it should be removed from the collage grid.
- The preview button should generate a preview of the collage based on the current images and layout.
- When a user selects a background color, the collage background should update accordingly.

## User Experience
- The file dropzone should be clearly visible and easy to use for uploading images.
- The grid layout should be responsive and adjust to different screen sizes.
- The drag-and-drop functionality should be smooth and intuitive for rearranging images.

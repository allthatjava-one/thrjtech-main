# Meme Generator Guide

Learn how to create shareable memes quickly using the THRJ Meme Generator. This guide walks you through uploading an image, adding and positioning text layers, and exporting your finished meme.

![Meme generator screenshot](/screenshots/meme-generator/meme-generator001.png)

## Quick Steps

1. Click the preview area or drag & drop an image to upload.
2. Use the `＋` button to add a text layer. Click a layer to edit its text.
3. Drag a text layer to position it over the image. Use the Advanced section to adjust font size and color.
4. Use Alt + Scroll (desktop) or pinch (mobile) to zoom and drag to pan the image inside the preview.
5. When finished, click **Download Meme** to save a PNG.

## Tips for Popular Meme Formats

- Classic top/bottom caption: add two layers, place one at the top and one at the bottom. Use a bold font size (e.g. 48+) and white text with a black stroke for high contrast.
- Single-line banner: add a single layer and center it horizontally. Reduce font size if it spans multiple words.
- Multi-line captions: press Enter inside a layer's textarea to create explicit line breaks. The editor preserves your line breaks.

## Best Practices

- Keep text short and punchy — memes work best with concise captions.
- Use high-contrast colors (white text + black stroke) to ensure readability over busy images.
- Preview at full size before downloading to make sure nothing is cropped.
